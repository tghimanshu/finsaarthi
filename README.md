# Finsaarthi

Finsaarthi is a static website for Finsaarthi Learning Solutions, an organization dedicated to promoting financial literacy and investor education in India. This project provides a web presence to showcase their mission, services, and strengths.

## About the Project

Finsaarthi aims to empower individuals and families with accurate financial knowledge. The website highlights their services, including:
- Financial Literacy and Education Programs
- Investor Education and Awareness Programs
- Financial Skill Development Programs

## Project Structure

The repository is organized as follows:

```
.
├── assets/                  # Static assets including CSS, JS, images, and fonts
│   ├── css/                 # Stylesheets (Bootstrap, custom styles, libraries)
│   ├── fonts/               # Font files
│   ├── images/              # Images used in the website
│   └── js/                  # JavaScript files (Main logic and libraries)
├── index.html               # Main HTML entry point
├── README.md                # Project documentation

└── finsaarthi.png           # Project banner/logo image
```

## Getting Started

Since this is a static website, you don't need a complex backend setup.

1.  **Clone the repository:**
    ```bash
    git clone <repository_url>
    ```
2.  **Open the website:**
    Simply open the `index.html` file in your preferred web browser.

## Usage

-   **Navigation:** Use the navigation bar to jump to different sections like "About Us", "Our Services", etc.
-   **Contact:** The footer contains a contact form (integrated with a static form service) and contact details.
-   **Animations:** The site uses WOW.js and Particles.js for visual effects.

## Technologies Used

-   **HTML5**
-   **CSS3** (Bootstrap 5, LineIcons, Animate.css)
-   **JavaScript** (Vanilla JS)
-   **Libraries:**
    -   Bootstrap
    -   GLightbox (Video gallery)
    -   CounterUp (Number animation)
    -   WOW.js (Scroll animations)
    -   Particles.js (Background particles)
